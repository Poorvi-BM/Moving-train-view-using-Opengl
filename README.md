# Moving Train View using OpenGL 🚆

This project is a graphical simulation built using OpenGL that displays a moving train scene. It creates a dynamic environment where objects like buildings, trees, and tracks move continuously to simulate the view from a running train.

## 🚀 Features
- Smooth animation of moving train view
- Real-time rendering using OpenGL
- Continuous scrolling background
- Simple and interactive visual experience

## 🛠️ Technologies Used
- C++
- OpenGL
- GLUT (for windowing and rendering)
